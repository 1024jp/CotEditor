Change preferences in CotEditor

Use CotEditor preferences to change options for the appearance and behavior of the editor and for how to handle documents.

In the CotEditor app on your Mac, choose CotEitor > Preferences, then click a preference pane:

  • General: Specify how to handle documents, saving, and more.
  • Window: Customize the default window appearance such as widow size or line number display.
  • Appearance: Set the appearance of documents, such as font, visibility of control characters, and the colors.
  • Edit: Specify how the editor behaves while editing.
  • Format: Set the default format for documents such as line ending type and text encoding, and to manage syntax styles.
  • File Drop: Choose how to insert a preset text when you drop a file onto the editor.
  • Key Bindings: Customize keyboard shortcuts to execute menu item commands, and to store some text snippets you use frequently along with your preferred keyboard shortcuts.
  • Print: Set the default style for document printing.
