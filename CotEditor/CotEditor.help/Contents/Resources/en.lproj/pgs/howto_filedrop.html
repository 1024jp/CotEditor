Drop files

File Drop is the feature to insert custom text based on the files dropped to the editor.

Perform File Drop

Drop files to the point to insert in the editor area.

You need to set the rule of the text to be inserted in advance. When no rule for the dropped file is found, the full path of the file will be inserted.

Define File Drop

To configure the behavior of File Drop, go to CotEditor > Preferences > File Drop.
