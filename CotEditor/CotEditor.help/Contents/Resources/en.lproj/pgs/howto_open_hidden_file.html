Opening hidden files

The checkbox “Show hidden files” in the open panel can toggle the visibility of hidden files in the panel.

You can also meke them visible as follows:

  1. While holding down the Option key, choose File.
  2. Select Open Hidden.
  • If you press the Option key while showing the File menu, Open will turn into Open Hidden.
  • Pressing Command-Option-O gives you the same result.
