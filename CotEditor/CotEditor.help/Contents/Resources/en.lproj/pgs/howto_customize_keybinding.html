Customize key bindings

You can fully customize CotEditor’s menu key bindings. Also, you can store text snippets which have your preferred key bindings assigned. For instance, you can insert “<br />” (with a line ending) by pressing Shift-Return. Go to Key Binbings pane in the preferences to create your own key bindings.

  • You cannot set the shortcuts that are already taken (e.g. shortcuts for your script files or menu items that don’t appear in the Menu Key Bindings list).
  • The shortcuts defined in the Script menu are not checked when assigned but will be automatically overriden.
