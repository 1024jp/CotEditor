Replacing with preset

CotEditor has a powerful replacement feature that also can use regular expression.

Options

OptionDescription
FindA string to search for. You cannot leave this field blank. When it is blank, the row will be ignored as it is invalid.
Replace WithA string to replace with. When this field is left blank, the matches will just be removed.
RE (Regular Expression)Search with regular expression.
IC (Ignore Case)Ignore character case on search.
DescriptionYou can use this field to leave a comment about the term. Description doesn't affect to replacement. This can be left blank.

Advanced Find Options

You can customize find behavior from the advanced options panel. The advanced options panel will be opened when you click the gear button in the replacement set panel.

See Find and Replace for details of each option item.
