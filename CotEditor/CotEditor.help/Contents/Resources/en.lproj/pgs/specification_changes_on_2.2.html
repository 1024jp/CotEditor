Specification Changes on CotEditor 2.2

CotEditor 2.2 includes some specification changes from the previous versions in order to conform to the Sandbox rules. If you've used the legacy version before, you may need some manual migrations.

Scripts Folder Location

Due to the Sandbox requirements, the scripts folder location moved from ~/Library/Application Support/CotEditor/ScriptMenu/ to ~/Library/Application Scripts/com.coteditor.CotEditor/. Users need to migrate their script to the new folder manually, since CotEditor doesn't have the write permission to the new location.
